# Beautiful TODO App

A simple, beautiful TODO app with filters, search, drag & drop and JSON import/export.  
Runs fully static on GitHub Pages.

## Features
- Add / remove / edit tasks
- Mark as done
- Filters (All / Active / Completed)
- Search
- Drag & drop reorder
- Import / Export JSON

## Run locally
Just open `index.html` in your browser.

## Deploy on GitHub Pages
Push this repository to GitHub and enable GitHub Pages from the repository settings.
